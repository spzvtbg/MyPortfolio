# 🌐 Personal Portfolio Website

> A public project documenting the complete lifecycle of building my personal portfolio website—from initial idea to production.

## About

This repository is more than a website project.

It documents the complete planning, design, development, deployment, and continuous improvement of my personal portfolio website while following professional software engineering and project management practices.

The objective is to create a modern portfolio website **and** demonstrate how a software project is managed from concept to production.

---

## Project Goals

- Build a modern and responsive portfolio website
- Present my professional profile and projects
- Document the complete project lifecycle
- Apply software engineering best practices
- Practice project management
- Learn and document architectural decisions
- Showcase technical and organizational skills
